Курсовой проект по объектно-ориентированному программированию

## Структура проекта

- **backend** — бэкенд на Java (Spring Boot, Maven):
  - `pom.xml` — зависимости и настройки Maven.
  - `src/main/java/com/cybersystems`:
    - `CyberSystemsApplication` — точка входа Spring Boot.
    - пакет `domain` — ООП-модель (уровни, профиль игрока).
    - пакет `service` — бизнес-логика (каталог уровней, управление профилем).
    - пакет `api` — REST-контроллеры и DTO для общения с фронтом.
- **frontend** — клиентская часть (игра и страницы):
  - **home** — главная страница игры (`index.html`).
  - **level-map** — карта уровней (`index.html`).
  - **level-1-1** — страница уровня 1.1 с блок-программированием (`index.html`).
  - **sandbox** — старые/прототипные версии уровней (например, старый вариант уровня 1.1).
  - В корне `frontend` лежат общие файлы: стили (`style.css`), скрипты (`game.js`, `blockly-setup.js`, и т.д.), а также `server.js` для локального запуска.

## Запуск фронтенда

1. Перейти в папку `frontend`.
2. Установить зависимости: `npm install` (если ещё не ставили).
3. Запустить сервер: `node server.js`.
4. Открыть в браузере `http://localhost:3000` — загрузится главная страница из `frontend/home/index.html`.

## Запуск бэкенда (Java)

1. Убедиться, что установлен JDK (рекомендуется 17) и Maven.
2. Перейти в папку `backend`.
3. Собрать и запустить приложение:
   - `mvn spring-boot:run`
4. Бэкенд поднимется на `http://localhost:8080`, доступные примерные эндпоинты:
   - `GET /api/levels` — список уровней.
   - `GET /api/profile` — текущий профиль игрока.
   - `POST /api/profile/progress` — обновить прогресс (тело: `{ "levelId": "1.1" }`).
   - `POST /api/profile/nickname` — обновить ник (тело: `{ "nickname": "Имя" }`).
